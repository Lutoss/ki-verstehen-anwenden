# Design-Varianten (Teil 1)

Fünf gestalterische Experimente derselben Inhalte von **Teil 1** — als
Vergleich verschiedener Layout- und Stilrichtungen. Der Kurs selbst läuft
über die Fassung in [`../kurs/teil-1-orientierung.html`](../kurs/teil-1-orientierung.html);
diese Varianten sind Ideengeber, keine gepflegten Hauptfassungen.

| Datei | Stil |
|-------|------|
| [`lehrbuch.html`](lehrbuch.html)     | Lehrbuch — ruhig, textorientiert |
| [`raster.html`](raster.html)         | Raster — kartenbasiertes Grid |
| [`arbeitsheft.html`](arbeitsheft.html) | Arbeitsheft — zum Mitmachen |
| [`neobrutal.html`](neobrutal.html)   | Neo-Brutalismus — kräftig, kontrastreich |
| [`terminal.html`](terminal.html)     | Terminal — Mono/Code-Ästhetik |
